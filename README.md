# Transparent Background Image Slider

A simple and interactive JavaScript project that compares two images using a draggable slider effect. As the user moves the mouse across the image, the transparent overlay reveals the original image underneath, creating a smooth before-and-after comparison experience.

## Features

* Interactive image comparison slider
* Smooth reveal effect using HTML, CSS, and JavaScript
* Responsive design for different screen sizes
* Lightweight and easy to customize
* Perfect for before/after image showcases

## Technologies Used

* HTML5
* CSS3
* JavaScript

## How It Works

1. A transparent image is displayed as the top layer.
2. The original image is placed beneath it.
3. Moving the mouse horizontally adjusts the width of the overlay container.
4. The slider line follows the cursor position, revealing more or less of the original image.
